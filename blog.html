What is the Document Object Model?

It's essentially the interface between your JavaScript code and the webpage, making it possible to create dynamic and interactive experiences.

how to select an element from the dom?
  1. 1. getElementById-- Selects an element by its unique ID.
  2. 2. getElementByClassName-- Selects an element by its ClassName.
  3. 3. getElementsByTagName--- Selects elements by their tag name. This also returns a live HTMLCollection.
  4. 4. querySelector--- Selects the first element that matches a CSS selector. This method is very versatile.
  5. 5. querySelectorAll--- Selects all elements that match a CSS selector and returns a static NodeList.
what is event delegation in the context of the dom, and why is it usefull?

Event delegation is a technique in the context of the DOM where I attach a single event listener to a parent element instead of adding individual listeners to multiple child elements. This allows me to manage events more efficiently.
